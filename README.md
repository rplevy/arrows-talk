# arrows-talk

Lightning Talk given at Bay Area Clojure Meetup. A quick tour of Clojure threading macros.

  * the basic core library arrows
    * ->

  * rplevy/swiss arrows
    * -<> ; diamond wand
    * -?<> ; nil-shortcutting
    * -!>, -!<> etc ; non-updating

  * new core library arrows in 1.5
    * cond->
    * as->
    * some-> ( formerly core.incubator -?> )

  * LonoCloud/synthread
    * ->/assoc
    * ->/in
    * ->/for
    * ->/when
    * ->/aside
    * ->/reset
    * ->/apply
