# arrows-talk

A quick tour of Clojure threading macros:

  * the basic core library arrows
    * ->

  * swiss arrows
    * -<> ; diamond wand
    * -?<> ; nil-shortcutting
    * -!>, -!<> etc ; non-updating

  * new core library arrows in 1.5
    * cond->
    * as->
    * when->
    * some->
     * formerly core.incubator -?>

  * Pallet/thread-expr

  * LonoCloud/synthread
